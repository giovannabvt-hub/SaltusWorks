# Imágenes del sitio

## `hero.jpg` — imagen principal del Home (¡importante!)

La homepage está preparada para mostrar automáticamente la obra del brief
(el árbol de la vida al amanecer con el wordmark "Saltus Works").

**Para activarla:** guardá esa imagen en esta carpeta con el nombre exacto
`hero.jpg`:

```
public/images/hero.jpg
```

- Formato recomendado: JPG o WebP, orientación horizontal (16:9), ~1920px de ancho.
- Mientras no exista el archivo, el hero muestra un gradiente cálido
  "noche → luz" que ya luce elegante por sí solo, así que el sitio nunca
  se ve roto.
- Esta misma imagen se usa como `og:image` para compartir en redes.
